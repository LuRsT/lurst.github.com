In the style of The Setup, this is my self-interview about my setup.

Who are you, and what do you do?

I'm Gil, a software developer of mostly web stuff.

What hardware do you use?

I have a T440s and a X230, I love thinkpads, I use the X230 mostly for being with me everywhere I go, the T440s is for those moments where portability is not essential (it still is a very light and small machine).

I also own a custom build machine but I don't count that towards my setup since it's basically just for gaming.

And what software?

I try to use cli software as much as I can, so I edit code in Vim (sometimes I use PyCharm with the vi plugin for some projects), tmux and a console and Firefox.

Most of the software I write is written with Python

What would be your dream setup?

A super sturdy and light laptop that could not only enable me to play games, but to also develop, wik
